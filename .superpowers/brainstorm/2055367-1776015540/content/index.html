技术方案

1. 数据驱动 (demo.xls)

  • • Python Backend: 使用 Pandas 读取 Excel 所有工作表。
  • • 数据转换: 将深度数据转换为 JSON 序列。岩性关键词（如"砂岩"、"白云岩"）自动映射到符号 ID。
  • • 组合岩性: 支持通过混合背景色 + Pattern 覆盖来实现组合岩性。

2. 可视化引擎 (Vite + React)

  • • CSS Grid 布局: 替代原有 Canvas 实现，确保文字可搜索、布局可响应且 1:1 还原 `demo.jpg`。
  • • SVG 曲线: 使用 `` 渲染高精度测井曲线，支持无限缩放。
  • • OSM 地图: 集成 MapLibre，显示测井井位，点击后通过路由平滑跳转至综合图件。
